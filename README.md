# rn-dynamic-app-icon

Programmatically change the app icon in React Native 

## Installation

```sh
npm install rn-dynamic-app-icon
```

## Usage

```js
import { multiply } from "rn-dynamic-app-icon";

// ...

const result = await multiply(3, 7);
```
